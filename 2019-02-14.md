## Añadir un usuario a un grupo y si no existen crear el usuario y el grupo
```PowerShell
gc .\crear.txt | %{
$grupo = $_.split(",")[0]
$usuario= $_.split(",")[1]
New-LocalUser $usuario -NoPassword
New-LocalGroup $grupo
Add-LocalGroupMember $grupo $usuario
}

Get-LocalGroupMember gi1
```
